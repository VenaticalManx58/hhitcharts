Все скрипты формата myscript_* используются в планировщике задач cron. Хранятся и запускаются в корневой папке хостинга.
Данные и графики обновляются каждый день, токен доступа к hh api раз в неделю
myscript_first, myscript_second, myscript_third обновляют соответственно первую ,вторую и третью часть профессий. Скрипт, обновляющий сразу все части работает довольно долго, поэтому на случай сбоя, удобнее их разделить
myscript_changetoken обновляет Access Token в config.py
